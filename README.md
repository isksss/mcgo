# mcgo

PaperMC and Velocity manager.

## Usage

1. **Create Default Config**: `mcgo init`
2. **Download Plugins**: `mcgo download`
3. **Run Server**: `mcgo run`

### Command

1. server command(ex: list, plugins): `list`
2. mcgo command: `mcgo!stop`
